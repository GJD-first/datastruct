# 树状数组

```c++
int lowbit(int x)
{
    return x&(-x);
}
```



### 更新

```C++
void update(int i,int val)
{
    while(i<=n)
    {
        tr[i]+=val;
        i+=lowbit(i);
    }
}
```



### 查询

```c++
int query(int i)
{
    int ans=0;
    while(i>0)
    {
        ans+=tr[i];
        i-=lowbit(i);
    }
    return ans;
}
```
